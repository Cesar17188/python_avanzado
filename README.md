# python_avanzado

Código manejo de python avanzado

## Tipos de datos con tipado fuerte

- Se debe crear un entorno virtual (venv)
  - py -m venv venv
  - iniciar el entorno virtual .\venv\Scripts\activate
- instalar la libreria mypy (para darle tipado fuerte)
  - pip install mypy
